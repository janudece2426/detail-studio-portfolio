# Detail Studio Portfolio

상세페이지 제작자/디자이너를 위한 프리미엄 포트폴리오 랜딩 사이트입니다.

## 실행

```bash
npm install
npm run dev
```

로컬 주소는 `http://localhost:3000`입니다.

## 주요 구조

- `src/components`: 섹션별 UI 컴포넌트
- `src/data`: 포트폴리오, 서비스, 프로세스, 문의 링크 데이터
- `src/app`: Next.js App Router 페이지와 전역 스타일

이미지와 링크는 `src/data` 폴더의 배열 데이터에서 쉽게 수정할 수 있습니다.
